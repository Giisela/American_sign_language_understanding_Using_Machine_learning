é necessario Python 3.7 e respetivo pip

instalar:
opencv
pip install numpy
pip install scipy
pip install sklearn
pip install seaborn
pip install opencv-python==3.7.x

para correr o projeto basta escrever no terminal 
python3 start_SVM_classifier.py -> no caso de querer SVM
python3 start_LOGISTIC_classifier.py -> no caso de querer Logistic Regression

e fará todo o processo automaticamente como se fosse um script

todas as informções estarão em
dados/labels/
dados/labels/result

e todas as imagens estão em
dados/iamges
dados/iamges/train -> no caso de treino
dados/iamges/test -> no caso de test
